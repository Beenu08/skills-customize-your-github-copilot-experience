# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice intermediate Python skills by working with strings, file input/output, and text manipulation to process and analyze text data.

## 📝 Tasks

### 🛠️ Word Frequency Counter

#### Description
Create a program that reads a text file, counts the frequency of each word, and writes the results to a new file.

#### Requirements
Completed program should:

- Read text from an input file (e.g., 'input.txt')
- Convert text to lowercase and remove punctuation
- Count occurrences of each word
- Write the word frequencies to an output file (e.g., 'word_counts.txt') sorted by frequency descending
- Handle file not found errors gracefully

### 🛠️ Text Search and Replace

#### Description
Write a script that searches for specific patterns in a text file and replaces them with new text.

#### Requirements
Completed program should:

- Read a text file
- Search for a user-specified word or phrase
- Replace all occurrences with a new word or phrase
- Write the modified text to a new file
- Display the number of replacements made

### 🛠️ Text Statistics Analyzer

#### Description
Build a program that analyzes text and provides various statistics.

#### Requirements
Completed program should:

- Read text from a file or user input
- Calculate and display:
  - Total number of characters
  - Total number of words
  - Average word length
  - Number of sentences (assuming sentences end with '.', '!', or '?')
- Handle empty input gracefully
